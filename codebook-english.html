Variable Overview

If the list is to be printed out, SoSci Survey will gladly open the list for you in a new window and without a frame.

For a printable overview of the questions, use the proof of the corresponding questionnaire.

qnr2sociodemographics 


The variable overview lists all the variables used in the survey project. The questionnaire and thus also the variables are sorted by category, position of question and item.

The overview also shows the possible expressions in addition to the variables. Further information about the values can be found in the manual.

Coding and return values

Variables for which no values have previously been stored are greyed out. These variables are not actively used until a value is specified in at least one completed questionnaire.

The overview is displayed in the active language version.


Questionnaire-internal data

The record contains the following additional variables in addition to your questions, unless you disable the option when downloading the record.

CASE Sequential number of the subject

REF reference, if such a link was added to the questionnaire

LASTPAGE Number of the page in the questionnaire that was last edited and sent

QUESTNNR ID of the questionnaire that has been edited

MODE Information, whether the questionnaire was started in the pretest or by a project member

STARTED Time at which the participant called the questionnaire

FINISHED Information on whether the questionnaire has been completed to the last page

TIME_001 ... Time a participant spent on a questionnaire page

Please note that you can not read out the questionnaire-internal variables with the value () function. For the interview number and reference, however, the PHP functions PHP function caseNumber () and PHP function reference () are available.

Details about the additional variables are in the manual: Additional variables in the data output
